# FakeStore Products App

React Native (Expo) application that fetches products from **FakeStore API** and displays them using **FlatList**.

## Features

- Fetch products from https://fakestoreapi.com/products
- useEffect + useState for API call and state management
- FlatList rendering
- Product card UI similar to provided design
- Loading indicator while fetching data
- Error handling
- Icons using @expo/vector-icons

## Technologies Used

- React Native
- Expo
- TypeScript
- FakeStore API
- FlatList
- @expo/vector-icons
